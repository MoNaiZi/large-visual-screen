# large-visual-screen
## 技术栈：
```
前端： Vue3+echarts+element-ui
后端： node.js+express+mongodb
```
## 前端，安装，运行，打包
```
yarn install
yarn serve
yarn build
```
## 后端
```
进入server目录
yarn install  
supervisor index 运行
```
